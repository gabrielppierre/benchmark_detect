# Deepsearch References

Esta pasta é reservada para materiais de contexto complementar do projeto, como:

- relatórios em Markdown;
- PDFs técnicos;
- notas de benchmark;
- referências externas relevantes para o dataset, protocolo ou modelagem.

Uso recomendado:

- manter aqui apenas documentos que ajudam a contextualizar decisões;
- preferir nomes de arquivo descritivos, com data quando fizer sentido;
- preservar os arquivos originais, sem editar PDFs de origem;
- não usar esta pasta para artefatos gerados da pipeline.

Quando houver conflito entre um material desta pasta e a documentação normativa do repositório, prevalecem os contratos e protocolos em `docs/`.
